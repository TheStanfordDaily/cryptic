# Cardinal Cryptic — Embed Guide

1. Open `generate.html` in a browser.
2. Fill in the fields:
   - **Clue** — full clue text, e.g. `Confused alien (5)`
   - **Answer** — letters only, e.g. `LIANE`
   - **Puzzle #** — increment by one each day
   - **Definition / Indicator / Fodder** — quote each part: `"confused"` or `"cat", "door"` for non-contiguous parts
   - **Explanation** — shown after the puzzle is solved, e.g. `Anagram of ALIEN gives LIANE.`
3. Click **Generate Embed Code**, then **Open preview ↗** to verify.
4. Click **Copy**, then paste into a **Custom HTML** block in WordPress.
